# 🍓 LuiFresh — Premium Organic Fruit Store

Welcome to **LuiFresh**, a high-end e-commerce interface designed with a focus on modern UI/UX and seamless performance. This project showcases a complete "Fruitery" shopping experience.

## 🚀 Live Demo
[**Click here to view the live site**](https://sava132.github.io/lui-fresh-store/) 
*(Замени ссылку на свою, когда активируешь GitHub Pages)*

## ✨ Key Features
- **Smart Product Grid:** Automatically adjusting layout for 8+ premium fruits.
- **Real-time Search:** Instant filtering of products using Vanilla JavaScript.
- **Advanced Shopping Cart:** - Add items with a single click.
  - Remove specific items from the basket.
  - Real-time total price calculation.
- **Premium Design:** Glassmorphism aesthetic with smooth CSS animations and backdrop filters.
- **Interactive Checkout:** A custom-built checkout flow with a success state.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Flexbox & Grid)
- **Logic:** Vanilla JavaScript (ES6+)
- **Icons:** [Lucide Icons](https://lucide.dev/)
- **Design Style:** Glassmorphism / Modern Dark Mode
